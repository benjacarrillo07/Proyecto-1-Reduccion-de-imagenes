# Proyecto-1-Reduccion-de-imagenes
Cuando se observa el cielo hay que transformar fotones a unidades digitales que podamos usar para construir imágenes. Este proceso es posible gracias a los 'Charged Couple Devices' (CCD), placas metálicas que liberan electrones cuando la luz del objeto de interés choca la placa. Los electrones son almacenados en el pozo potencial generado entre los nodos de los CCD, que poseen varias celdas. Los píxeles son un conjunto de celdas que almacenan electrones. Una vez hecha la observación, un sensor cuenta los electrones almacenados en cada píxel y le asigna un valor de intensidad digitalmente (ADU).
