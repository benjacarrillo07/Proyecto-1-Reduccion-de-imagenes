# Proyecto-1-Reducci-n-de-im-genes
Las imágenes de ciencia son obtenidas mediante CCDs, instrumentos que miden el efecto fotoeléctrico producido por la luz, que es interpretados digitalmente como ADUs. Otros efectos añaden electrones a la medición, generando ADUs que no provienen del objeto observado. La calibración y reducción se encarga de eliminarlos y dejar una imagen limpia.
